La regola d'oro: zero dati inventatiThe golden rule: zero invented dataЗолотое правило: никаких выдуманных данных

Il riconoscitore estrae un servizio solo se trova la frase sul sito; poi il programma ri-verifica da solo che quella citazione esista davvero nel testo (confronto esatto). Se non c'è → il dato è scartato, mai indovinato. Nessun sito → punteggio 0.The recognizer extracts a service only if it finds the sentence on the site; then the program re-verifies on its own that the quote truly exists in the text (exact match). If not → the data is discarded, never guessed. No website → score 0.Распознаватель извлекает услугу, только если находит фразу на сайте; затем программа сама перепроверяет, что цитата действительно есть в тексте (точное совпадение). Если нет → данные отбрасываются, не угадываются. Нет сайта → оценка 0.
